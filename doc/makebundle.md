# Requirement

- vcpkg
- python3
- Visual Studio 2017 and above

# Steps

1. install gtk/gtkmm package.
```bash
vcpkg install gtkmm
```
2. change settings in `gtkbundle.py`.
```bash
vi gtkbundle.py
```
3. bundle it!
```bash
python gtkbundle.py
```
4. zip it!
5. publish it!
